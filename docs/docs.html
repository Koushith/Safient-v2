Introduction
SafientUI is a React UI Framework for building ambitious products on the web.
Installation Guide
SafientUI can be installed by either NPM or Yarn or using CDN link, using the following command(s). You should also have react@16.8.0 and react-dom@16.8.0 or higher versions. 
                        
            <link rel="stylesheet" href="https://safientui.netlify.app/css/components.css">
